概要
====

時雨堂製品紹介と時雨堂座談会のオフラインイベントです。

昼の部と夜の部の２つに分かれています。
時雨堂製品に興味がある企業の方は昼の部へ、時雨堂座談会を聞きたい人は夜の部へどうぞ。

昼の部は時雨堂の主力製品 WebRTC SFU Sora と Media Pipeline Hisui の紹介を行います。
また余裕があれば QUIC 関連の話題も入れたいと思っています。

夜の部はゲストに `ラムダノート株式会社 <https://www.lambdanote.com/>`_ `鹿野さん <https://x.com/golden_lucky>`_ を呼んで、IT 技術出版の話題を中心に時雨堂座談会を開催します。
